Requerimientos 
Java versión 8

Instrucciones
Utilizando la consola como administrador sigua los siguientes pasos: 

1. Ubicarse en la capeta donde se encuentra la entrega luego en la carpeta “src\ ejercicio-psp2” Ejemplo D:\Proyectos\UniAndes\PSP2\src\ejercicio-psp2
2. Ejecute el siguiente comando: mvn package
3. Ejecute el siguiente comando: java -cp target/ejercicio-psp2-1.0.jar edu.uniandes.ecos.PSP2.App
