Instrucciones
Utilizando la consola como administrador sigua los siguientes pasos: 

1. Ubicarse en la capeta donde se encuentra la entrega luego en la carpeta “src\program4” Ejemplo D:\Proyectos\UniAndes\PSP2-Program4\src\program4
2. Ejecute el siguiente comando: mvn package
3. Ejecute el siguiente comando: java -cp target/program4-1.0.jar edu.uniandes.ecos.PSP2.App.App
